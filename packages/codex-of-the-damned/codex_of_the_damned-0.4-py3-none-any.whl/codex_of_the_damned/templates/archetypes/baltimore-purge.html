Baltimore Purge

{% endblock %} {% block archetype %}

Highlights

This is a special deck using {{ card("Montano") }} to play {{ card("Baltimore Purge") }}, then {{ card("Freak Drive") }} to get out of torpor and play {{ card("Graverobbing") }} to permanently steal opponents vampires. {{ card("Mind Rape") }} is added to the mix for additional vampire thefts. {{ card("Heidelberg Castle, Germany") }} is used to take the blood of stolen vampires and give it to {{ card("Montano") }}, fueling the costly combination.

It is a dreadful combination and, therefore, quickly attracts the table attention. This archetype is seldom played and has not yet won any major tournament.

The deck packs quite a lot of $ stealth considering {{ card("Montano") }} already has an included stealth bonus: indeed, getting blocked somewhere suring the combo can be deadly, so the deck has to make sure it does not happen.

The entire combat module is defensive, using {{ card("Shadow Body") }} and {{ card("Obedience") }}.

Tips & Tricks

{{ card("Zillah's Valley") }}, {{ card("Dreams of the Sphinx") }} and {{ card("Information Highway") }} are used to get {{ card("Montano") }} out faster.

{{ card("Enkil Cog") }} lets {{ card("Montano") }} bleed for 3 and act during the other players turns. Playing a {{ card("Baltimore Purge") }} during the predator's turn makes the whole combination easier.

{{ card("Renewed Vigor") }} is used at inferior f to help get {{ card("Montano") }} out of torpor when he plays {{ card("Baltimore Purge") }}.

{{ card("Eluding the Arms of Morpheus") }} provides additional unlock: it can be used to unlock during an opponent's turn and, if {{ card("Enkil Cog") }} is into play and the block does not happen, a new action can be performed at the end of the opponent's turn.

Variants

There are only minor variations to this archetype. {{ card("Govern the Unaligned") }} and {{ card("Perfectionist") }} are often included.
